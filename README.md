## 卫星约束

1. 最低卫星轨道为1.2
2. 卫星轨道之间的距离不得小于0.2
3. 每一个卫星必须有一个唯一的COSPARID，总长度为6位，由两位国家编码加上四位数字组成
4. 每一个卫星都有一个容易识别的名称，可以由所属国家命名
5. 卫星可以有两种状态，一种状态是处于激活可用状态，另一种是处于封锁非激活状态，但卫星任然在固定轨道飞行，只是处于不可用状态
6. 卫星报废后，应该及时被删除，腾出轨道空间
